### 一、题目描述
给你一个整数数组 nums，返回 数组 answer ，其中 answer[i] 等于 nums 中除了 nums[i] 之外其余各元素的乘积 。
题目数据 保证 数组 nums之中任意元素的全部前缀元素和后缀的乘积都在  32 位 整数范围内。
请不要使用除法，且在 O(n) 时间复杂度内完成此题。

示例 1:
> 输入: nums = [1,2,3,4]
>
> 输出: [24,12,8,6]

示例 2:
> 输入: nums = [-1,1,0,-3,3]
>
> 输出: [0,0,9,0,0]

提示：
- 2 <= nums.length <= 105
- -30 <= nums[i] <= 30
- 输入 保证 数组 answer[i] 在  32 位 整数范围内

**进阶**：你可以在 O(1) 的额外空间复杂度内完成这个题目吗？（ 出于对空间复杂度分析的目的，输出数组不被视为额外空间。）


### 二、解题思路

这道题的最优解法是**左右乘积数组法（也叫前缀后缀乘积法）**，核心思想是：
- **先计算每个元素左侧所有元素的乘积（前缀乘积）**。
- **再计算每个元素右侧所有元素的乘积（后缀乘积）**。
- **最终每个位置的结果 = 左侧乘积 × 右侧乘积**。

### 三、题目实现
```javascript
var productExceptSelf = function(nums) {
    if(nums.length === 0) {
        return []
    }
    let answer = new Array(nums.length).fill(1);
    // 左侧的乘积
 
    // 第一步：正向遍历，计算每个元素的左侧乘积，存入answer
    let left = 1; // left表示当前元素左侧所有元素的乘积，初始为1（第一个元素左侧无元素）
    for(let i = 0; i < nums.length; i++) {
        answer[i] = left; // 先把左侧乘积存入结果数组
        left *= nums[i]; // 更新left，为下一个元素的左侧乘积做准备
    }
    // 第二步：反向遍历，计算每个元素的右侧乘积，并与左侧乘积相乘得到最终结果
    let right = 1; // right表示当前元素右侧所有元素的乘积，初始为1（最后一个元素右侧无元素）
    for(let i = nums.length -1; i>= 0; i--) {
        answer[i] *= right; // 左侧乘积已经存在answer[i]中，现在乘上右侧乘积
        right *= nums[i]; // 更新right，为下一个元素的右侧乘积做准备  
    }
    return answer
};

```
